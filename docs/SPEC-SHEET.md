# Build Specification Cards

Compiled from documents in the archive. Each fact is traceable to source documents in data/master-record.json.

## Engine
- Platform: VW Type 4 conversion (upright, in Type 1 chassis)
- Displacement Cc: 1971
- Displacement Note: 94mm bore x 71mm stroke = 1,971cc, referred to as '2000cc' in build notes
- Bore Mm: 94
- Stroke Mm: 71
- Compression Ratio: 8.6:1 static
- Deck Height In: 0.028
- Chamber Volume Cc: 61
- Head Work: Valve job, opened for 94mm, flycut .050, spark plug inserts (one head has 2nd-oversize insert), Type 4 rocker spacer kit
- Camshaft: Engle (grind number unknown, data gap)
- Carburetion: Dual 44mm Webers (exact model unknown, presumed IDF, data gap). Dual Weber setup confirmed in continued service via 2x rebuild kits purchased 2018-05-31 (NV Automotive).
- Air Filtration: K&N oiled gauze implied by recharge kit purchase (verify)
- Fuel Pump: Mechanical pump blocked off (blockoff plate in Raby kit); electric pump presumed (verify)
- Oil Filtration: Type 4 stock spin-on filter
- Charging: Alternator on Raby billet stand with integrated oil filler
- Spark Plugs: Bosch W8DC (4 purchased 2015-05-21, Butch's Speed Shop)
- Head Work Date: 2013-08-13, NV Automotive (Bill Shapley), Las Vegas, invoice 125, $379.67
- Flywheel: Lightened flywheel ($90 per restorer's cost ledger, DOC-043)
- Bottom End Cost Record: DOC-043 itemizes: 94mm pistons/cylinders $169, crankshaft $60, rod/main/cam bearings $90, lifters $30, rockers $90, gasket set $25, alternator $90, oil pump milling $10, shroud assembly $1000 (matches DTM Stage III).
- Dtm Kit Purchase: 2013-07-29, Raby Enterprises receipt #1878 (sales order #94), shipped to Hurricane UT
- Sources: DOC-001, DOC-003, DOC-004, DOC-007, DOC-030, DOC-043

## Cooling
- System: Jake Raby DTM (Down The Middle) Stage III
- Description: Upright fiberglass shroud system routing cooling air evenly to all four cylinders, allowing Type 4 installation in a Type 1 vehicle without cutting the engine bay
- Components: Stage III shroud, oil cooler adaptor kit, 6 pc waterjet surround tin, air sealing kit, billet conversion pulley, DTM fan belt, right air hose, alt/gen stand mount bar
